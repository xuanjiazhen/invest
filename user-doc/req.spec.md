# 仓颉语言资料部署状态调研


当前仓颉语言作为鸿蒙应用开发的第二门语言，仓颉语言的开发指南、工具链指南、标准库API参考等相关的资料文档，在部署发布给用户的时候有一些选择：
- 仓颉语言相关的开发指南、工具链指南、标准库API参考等资料内容，从语言官网复制一份并适配放到鸿蒙应用开发的统一官网上
- 仓颉语言相关的开发指南、工具链指南、标准库API参考等资料内容仅存在语言官网，鸿蒙应用侧官网仅放简单介绍，然后链接至仓颉语言官网
- 仓颉语言相关的开发指南、工具链指南等资料内容仅存在语言官网，鸿蒙应用侧官网仅放简单介绍，然后链接至仓颉语言官网，标准库API在语言官网和鸿蒙应用官方都放一份
