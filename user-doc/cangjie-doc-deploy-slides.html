仓颉当前状态

  • 双站并存：语言站 + 平台站
  • 语言站侧重语言本身（语法、标准库、工具链）
  • 平台站侧重鸿蒙应用开发（Kit 指南、实践、FAQ）
  • 标准库 API 位置待最终确定

关键事实

  • 仓颉与鸿蒙同属华为
  • Swift 语言指南完整书籍在 docs.swift.org，平台站仅外链
  • Kotlin 语言指南及标准库均在语言站，平台站仅外链
  • ArkTS 全部统一于平台站
