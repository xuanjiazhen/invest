# Invest

使用 AI 辅助进行技术调研总结与各种 AI 尝试想法的仓库。

## 浏览

访问 [index.html](index.html) 查看所有交互式演示文稿（slides）。

## 结构

每个调研主题独立一个目录，包含需求描述、调研报告和交互式 HTML 幻灯片。

```
.
├── AGENTS.md              # AI 辅助编码规范
├── index.html             # Slides 索引首页
├── {topic}/               # 各调研主题目录
│   ├── req.spec.md        # 原始需求
│   ├── report*.md         # 调研报告
│   └── *-slides.html      # 交互式演示
└── output/                # Skill 运行产物（不提交）
```

## 协作

AI 改动前请先阅读 [AGENTS.md](AGENTS.md)。
