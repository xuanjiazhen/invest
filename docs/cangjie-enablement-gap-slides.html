🟡 第二阶段：补齐互动

  • P1 上线 CodeLab —— 3-5 个（Hello World → ArkUI → 网络 → 数据库 → 互操作）
  • P1 上线视频课程 —— 入门 + ArkUI 开发
  • P1 发布 cjlint 编码规范为官网独立文档
  • P1 独立 Sample 项目下载

🟢 第三阶段：深度运营

  • Release Notes 常态化
  • 社区仓颉专区运营
  • API 参考 Kit 覆盖率对齐 ArkTS
